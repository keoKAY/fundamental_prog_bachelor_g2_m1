## RULE FOR NAMING VAR
1. cannot start with number 
2. cannot used keyword 
3. cannot use with other symbol beside (`_`)
4. cannot contains white space 

Convention for naming variable 
- short and concise
- use camelCase Writing Style for variable and method 


lowercase = firstname 
uppercase = FIRSTNAME 
snakecase = first_name 
camelcase = firstName; dateOfBirth,first
pascalCase = FristName , DateOfBirth
