## NOTE 

Parent Class, Base Class , Super Class  
Child Class , Derived Class , Sub Class 

**Inheritance**: 
A way for a one class to get the properties (code) from another class 
such as (data member, function members ... )

Parent Class provide the inheritance for Child
Child Class recieve the inheritance from Parent 

- **Advantages**: Code Reusability ( No need to repeat the writing the same class over again and again )
